BiovizJS Mouse and Keyboard Controls

In order to move objects on the scene, we provide mouse and keyboard controls in BiovizJS. You must know that controls are moving objects like a full scene, a molecular structure or a molecule and are not moving the camera. This enables to use 3D mouses to interact with BiovizJS (feature coming soon).

Here are the different mouse and keyboard bindings :

Mouse Controls

  • Double Left Click :
    Object selection, allowing to move the selected object
    Note : When double clicking on no object, you select the whole scene

  • Left Mouse Press :
    Rotate the selected object

  • Right Mouse Press :
    Translate the selected object

  • Right Click :
    Open BiovizJS context menu

  • Wheel Up/Down :
    Move selected object backward / forward

  • Ctrl + Wheel Up/Down :
    Increase / decrease eye separation for 3D effects (stereoscopy / anaglyph)

Keyboard Controls

  • F :
    Enable / Disable BiovizJS full-screen

  • L :
    Show / Hide color mapping legend

  • M :
    Show / Hide measure management tool

  • R :
    Show / Hide representation management tool
