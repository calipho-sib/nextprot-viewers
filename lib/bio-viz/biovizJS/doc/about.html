About BiovizJS

History

  • v0.2-beta :
    • Usage of Promises for asynchronous tasks
    • Usage of webworker to load PDB / mmCIF files
    • Simplified ContextMenu
    • Added fog on the scene
    • Added clipping planes
    • Added new management widgets (Representations, Color Mapping, Measurement)
    • Added hydrophobicity color mapping
    • External BVS Jobs bindings with BiovizJS
  • v0.1-beta :
    • Mouse selection events
    • Better camera management (zoom, near, far)
    • Added highlight feature
    • Usage of an IFrame to embed BiovizJS into a web page
  • v0.4-alpha :
    • Added a ContextMenu to manage scene by right click
    • Added a script player to handle with script by UI
    • Improved Javascript computation performances
  • v0.3-alpha :
    • Full script language support
    • Added a script engine to manage scripts
  • v0.2-alpha :
    • Alpha BiovizAPI to access and modify data by strings
    • Added external widget using the API
    • Improved rendering performance and CPU usage
  • v0.1-alpha : First alpha release of BiovizJS
