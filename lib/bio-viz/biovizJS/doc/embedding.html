Deploy BiovizJS

Requirements

BiovizJS requires JQuery 1.9 or above and JQuery-UI 1.11.4 or above on your website distribution folder.

Installation

Install the provided BiovizJS distribution folder in a folder accessible from the website root folder. We will name it's path on the system as following : BIOVIZ_PATH.

Please note it's important that BiovizJS distribution folder must be placed on the same domain as your website. Indeed, BiovizJS is using an IFrame to be deployed into your website, and you want to avoid Origin Policy errors.

Deploy to your Website

The following example shows you how you can deploy BiovizJS into a simple website : BiovizJS Simple Example.

First, you must declare a specific container for BiovizJS into your web page as on the following example :

<!DOCTYPE html>
<html lang="en">
  <head>
    ...
    <title>Bioviz App</title>
    ...
  </head>
  <body>
    ...
    <div id="bioviz">
    </div>
    ...
  </body>
  <script type="text/javascript" src="BIOVIZ_PATH/BiovizJS.js"></script>
</html>

As you can see, at the end of the page you must include the BiovizJS script located in BIOVIZ_PATH that allows to use the library.

Then, you have to initialize BiovizJS with custom settings in order to deploy it into your website :

<script type="text/javascript">
    var biovizWidget;

    // Set bioviz options
    var options = {
      structureToLoad: '4MMA'
    };

    $(document).ready(function() {
        // Launch bioviz app
        biovizWidget = $("#bioviz").bioviz(options);
    });
</script>

In this example, BiovizJS will directly load the PDB structure 4MMA, when the 3D context of the application is ready.

BiovizJS widgets are based of JQuery-UI Widgets Factory, please look here for further information.

Finally, if you're using BiovizJS for testing purpose on a local machine, you will have to start a simple HTTP server to be able to load the application. Please call this shell command on the root folder of your website (or on the BIOVIZ_PATH/example folder of BiovizJS to load the demo page) :

python -m SimpleHTTPServer

To access to the application, you have to look at localhost:8000 in your web browser. You will then be able to visualize the 4MMA structure from the PDB provided by RCSB into your screen.

Advanced Deployment

You may want to use external widgets to interact with BiovizJS. That's why we provide you an API, allowing a read / write access to the application. In order to be able to interact with BiovizJS, you must add :

  • A function that setup your environment
  • A binding to the "ready" event of Bioviz to start this function
<script type="text/javascript">
    var biovizWidget;

    // Setup onReady function (create external widgets)
    var onReady = function() {
      // For instance, add an external tool bar that allows
      // to modify representations of the objects of the scene
      var toolbarWidget = $("#bioviztoolbar").bioviztoolbar({bioviz: biovizWidget});

      // Callback to update this widget when Bioviz notifies a requestWidgetUpdate
      biovizWidget.bioviz({
        requestWidgetUpdate: function(event) {
          toolbarWidget.bioviztoolbar("instance").updateHTML(event.data);
        }
      });
    };

    $(document).ready(function() {
        // Launch Bioviz application with options
        ...

        // Bind to application ready, allowing to start external widgets
        biovizWidget.bioviz({
          ready: function(event) {
            onReady();
          }
        });
    });
</script>

For further information, please look at external widgets code examples provided on the BiovizJS Simple Example.
